# KintoneMajorToMinorItem

## Kintone 大項目から小項目の絞り込みプラグイン

## 1.概要

良くある別アプリで設定した大項目中項目小項目を絞り込みするプラグインです。

大項目から小項目の項目数に制限はありません。
